You are about to enter the
Simon's Game

The rules are pretty simple.
You are displayed four colours {green, red, yellow and blue}

  • Click on any key to start the game

  • Computer will blink a colour and you have to press it.

  • With each level a new color would be added to the sequence.

  • Remember : You have to press the correct sequence each time or you fail.


  • If you got a wrong color you can always restart by pressing any key on the keyboard.

Have Fun Playing!!
